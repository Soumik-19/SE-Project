# Software Project Management Tool

## Description
This is a Software Engineering project developed to help manage software projects efficiently. The application provides project management features such as task organization, requirement management, and effort estimation.

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- Firebase Firestore

## Features
- User Authentication
- Dashboard
- Kanban Board
- Requirements Management
- COCOMO Effort Estimation

## Project Structure

```
backend/
frontend/
docs/
package.json
```

## Installation

1. Clone the repository:
   git clone <repository-link>

2. Install dependencies:
   npm install

3. Start the application:
   npm start
