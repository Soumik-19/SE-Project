Design Principles Applied

▸
Modular Design — Each module (auth, kanban, COCOMO, testing) is a self-contained component with its own logic and UI.
▸
Information Hiding — Internal module logic is encapsulated; only public interfaces are exposed.
▸
Abstraction — COCOMO logic is abstracted behind a clean input/output interface.
▸
UI Design — Consistent design language with accessible colors and responsive layout.
